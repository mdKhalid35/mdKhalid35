## 🚀 Projects

### Enhancing Brain MRI Visualization Using DC-GAN
Developed a Deep Convolutional GAN model to generate synthetic brain MRI images to improve dataset diversity.

Tech Used:
PyTorch, Deep Learning, GANs

Key Highlights:
• Implemented DC-GAN architecture  
• Integrated DenseNet for feature learning  
• Addressed mode collapse challenges

---
### Aspect-Based Sentiment Analysis on Tourism Reviews
Built an NLP pipeline to analyze sentiments for specific aspects in tourism reviews.

Tech Used:
Python, NLP, Transformers, Machine Learning

Key Highlights:
• Developed an aspect-based sentiment model using the Mistral-7B transformer achieving 91% accuracy  
• Combined lexicon-based aspect extraction with transformer-based sentiment classification  
• Handled complex contexts such as sarcasm, negation, and contextual expressions

---

### Vision Assistance for the Visually Impaired
A real-time system that detects text and objects and converts them into speech to assist visually impaired individuals.

Tech Used:
Python, OpenCV, PyTesseract, Flask, pyttsx3

Key Highlights:
• Real-time text detection and speech feedback  
• Multithreaded speech pipeline  
• Web interface for interaction
